# Snake Game

[![Язык](https://img.shields.io/badge/Язык-JavaScript-yellow)]()
[![Тип](https://img.shields.io/badge/Тип-браузерная_игра-blue)]()

Классическая игра «Змейка» на чистом JavaScript без внешних библиотек.

## Что это за игра

Простая аркада, где игрок управляет змейкой, которая:
- двигается по полю;
- собирает «еду» (красный круг);
- растёт при поедании еды;
- проигрывает при столкновении с собственным телом или границами поля.

## Особенности реализации

- **Чистый JavaScript** — без jQuery, React и других фреймворков.
- **Простые правила** — понятно даже новичкам.
- **Счётчик очков** — отображает количество съеденной еды.
- **Управление** — стрелками клавиатуры.

## Как запустить

### Вариант 1. Онлайн (через GitHub Pages)

1. Перейдите по ссылке: [valera-rrr.github.io/snake-game/](valera-rrr.github.io/snake-game/)


### Вариант 2. Локально (с вашего компьютера)

1. **Скачайте проект**:
- Откройте репозиторий: [https://github.com/valera-rrr/snake-game](https://github.com/valera-rrr/snake-game)
- Нажмите зелёную кнопку `Code` → `Download ZIP`.

2. **Распакуйте архив** в любую папку на вашем компьютере.


3. **Откройте файл** `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).


4. **Начните игру** — нажмите «Старт» на экране.


## Управление

- **Стрелка вверх** — движение вверх.
- **Стрелка вниз** — движение вниз.
- **Стрелка влево** — движение влево.
- **Стрелка вправо** — движение вправо.

## Элементы интерфейса


- **«Старт»** — кнопка начала новой игры.
- **«Счёт: X»** — текущий счёт (количество съеденной еды).
- **Игровое поле** — область, где двигается змейка.
- **Красная точка** — еда для змейки.

## Технические детали


- **HTML5** — структура страницы.
- **CSS3** — стили и адаптивная вёрстка.
- **JavaScript** — логика игры (движение, коллизии, генерация еды).
- **localStorage** — сохранение лучшего результата между сеансами.
